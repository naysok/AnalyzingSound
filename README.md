### 解析にかける音域  

いまのところ、matplot でプロットしている横軸、0~2756 にしているが（拾ったコードがこうだったので）、あとで情報として使える音域を調べてみる。

Wikipedia の 聴覚 によると

>可聴域  
ヒトでは通常、下は20Hz程度から、上は（個人差があるが）15,000Hzから20,000Hz程度までの鼓膜振動を音として感じることができ、この周波数帯域を可聴域という。可聴域を超えた周波数の音は超音波という。
